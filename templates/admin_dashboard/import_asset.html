CSV Format Guide

Upload your assets using a CSV file. Ensure your file follows the structure below exactly. 

asset_id,name,description,category,sub_category,department,custodian,location,acquisition_date,cost,residual_value,useful_life_years,depreciation_method
AST-001,Laptop Dell XPS,Developer laptop,Electronics,Computers,IT,John Doe,Johannesburg,2024-01-10,25000,5000,5,Straight Line
  • • asset_id – Optional unique ID (auto-generated if empty)
  • • name – Asset name
  • • description – Required description
  • • category – Main category
  • • sub_category – Optional sub grouping
  • • department – Owning department
  • • custodian – Assigned user
  • • location – Physical location
  • • acquisition_date – Format: YYYY-MM-DD
  • • cost – Purchase cost (decimal)
  • • residual_value – Salvage value
  • • useful_life_years – Asset lifespan
  • • depreciation_method – e.g. Straight Line
